How This Website Was Built

SafetyUsingAI.org was built using modern web development technologies designed to create clean, responsive, and interactive websites. These tools allow the website to work seamlessly on computers, tablets, and smartphones.

HTML

HTML provides the basic structure and content of the website, organizing all text, images, and links into a well-organized format.

Tailwind CSS

Tailwind CSS is a utility-based framework that helps style the website beautifully and ensures it looks great on all devices, from phones to computers.

🎉 Fun Fact

This project consists of 77.7% HTML and 22.3% CSS

Development & Deployment

Version Control with GitHub

All progress on this website was carefully saved and tracked using GitHub, a platform for version control. This ensures that every change made to the code is documented and can be reviewed or reverted if needed, making collaboration and code management seamless.

Deployed with Vercel

This website is deployed and hosted on Vercel, a modern platform built for fast, reliable web applications. Vercel provides automatic deployments whenever changes are pushed to the repository, ensuring the website stays up-to-date with the latest improvements and features.

Our Mission

SafetyUsingAI.org was created with one important goal: to help students understand artificial intelligence and use it safely and responsibly.

Educate

We provide clear, accurate information about what AI is, how it works, and what it can do for society and education.

Inform

We help students understand both the benefits and risks of artificial intelligence so they can make informed decisions.

Empower

We equip students with practical safety tips and knowledge to use AI responsibly and protect themselves online.
